wb_PatternGenerator

Digital pattern generator

Generates an user defined digital pattern. Number of pattern output bits and memory size is defined at design time. The pattern is written sequentially, the number of writes determines the pattern length.

  • ENABLE [read/write]: Enable
    Enable trigger.
  • LOAD [read/write]: Load pattern
    Enable writing new digital pattern to memory.
  • STOP [write-only]: Stop pattern
    Stops and resets current pattern on writing 1.
  • SOFTTRIGGER [write-only]: Soft trigger
    Generates a trigger on writing 1.
